## TOP 10 FAILLES CRITIQUES

### 🚨 1. Sitemap JAMAIS soumis dans GSC (impact maximal)

GSC ne connait QUE 3 sitemaps obsoletes :

| Sitemap | Soumis le | Status |
|---------|-----------|--------|
| `https://www.allo-maison.ma/sitemap_index.xml` | 2020-12-15 | 5 warnings |
| `https://www.allo-maison.ma/sitemap.xml` | 2018-04-02 | **2 errors** |
| `http://www.allo-maison.ma/page-sitemap.xml` | 2018-03-22 | 4 warnings |

Le nouveau sitemap `https://allo-maison.ma/sitemap.xml` (141 URLs, tous 200) **n'a JAMAIS ete soumis**.

**Consequence** : Google decouvre les pages via liens uniquement = crawl lent + aucune priorite. C'est 95% de la raison pour laquelle 135 pages sont "URL is unknown to Google".

**Action immediate** : Soumettre via GSC UI ou API :
```
POST /webmasters/v3/sites/sc-domain:allo-maison.ma/sitemaps/https:%2F%2Fallo-maison.ma%2Fsitemap.xml
```

### 🚨 2. Indexation : 4.3% seulement (6/141)

URL inspection live (2026-04-17) :

| Etat | Nombre | % |
|------|--------|---|
| Submitted and indexed | 6 | 4.3% |
| URL is unknown to Google | 135 | 95.7% |
| Errors | 0 | 0% |

**Pages indexees** :
- `/` (homepage)
- `/plombier-tanger`
- `/bricoleur-casablanca`
- `/jardinier-casablanca`
- `/technicien-informatique-casablanca`
- `/cuisiniere-casablanca`

**Pages NON indexees** (parmi 135) :
- Toutes les pages statiques : `/services`, `/tarifs`, `/contact`, `/a-propos`, `/blog`, `/comment-ca-marche`, `/garantie`, `/confidentialite`
- Toutes les pages `/urgence/*/*` (18 URLs, piliers de la strategie)
- 108 des 114 pages service-city
- 5 des 6 pages `/plombier-*`

**Note** : La memoire projet datee du 2026-04-13 affirme "141/141 indexed" — c'etait une mesure de **submission** via Indexing API, pas d'**indexation reelle**. L'Indexing API est officiellement reservee a JobPosting + BroadcastEvent, Google ignore largement pour le reste.

### 🚨 3. Schema fake review (risque penalite manuelle)

Pages service-city contiennent :
```json
"aggregateRating":{"ratingValue":4.9,"reviewCount":2100,"bestRating":5,"worstRating":1}
```

**Probleme** : 2 100 avis affiches en schema **sans aucun avis visible sur la page** = violation directe des Google Structured Data Guidelines. Declenche potentiellement une **manual action "Spammy structured markup"** qui fait disparaitre TOUT les rich results du domaine.

**Action immediate** :
- Retirer `aggregateRating` de TOUTES les pages tant qu'on n'a pas d'avis clients verifiables affiches
- OU implementer un vrai systeme d'avis avec `Review` schema individuels

### 🚨 4. Pages urgence : contenu de basse qualite visible

Meta description `/urgence/plombier/casablanca` :
> "Besoin d'**un plomberie** en urgence **a** Casablanca ? Intervention en moins de 30 minutes. **Professionnels verifies** disponibles 24h/24 et 7j/7."

3 problemes en 1 phrase :
1. **Erreur de genre** : "un plomberie" = feminin au masculin. Devrait etre "d'un plombier"
2. **Accents manquants** : "a Casablanca" → "à Casablanca", "verifies" → "vérifiés"
3. **Mot-cle manque** : "plomberie" au lieu de "plombier" (le mot-cle cible)

Idem pour electricien/serrurier × 6 villes = 18 pages avec du contenu qui signale "auto-genere bas de gamme" a Google. **Zero impressions urgence en 4 jours** le confirme.

### 🚨 5. Schema urgence : squelettique

Pages `/urgence/plombier/casablanca` n'ont QUE :
- FAQPage
- BreadcrumbList
- Question / Answer / ListItem

**Manquent** (critiques pour ranking urgence) :
- `LocalBusiness` ou `EmergencyService` (requis pour pack local)
- `Service` avec `areaServed`, `hoursAvailable` (24/7)
- `Offer` avec `priceRange`
- `GeoCoordinates`

Comparaison avec mesdepanneurs.co (SERP #1) qui implemente tous ces types.

### 🚨 6. Trafic encore sur ancien domaine www/http

| URL GSC | Clicks | Impressions |
|---------|--------|-------------|
| `http://www.allo-maison.ma/` | **5** | **314** |
| `http://allo-maison.ma/` | 0 | 7 |
| `https://allo-maison.ma/` | 0 | 1 |

**96% des impressions et 100% des clics sont servis sur l'ancienne URL http+www.** Google sert l'ancienne version en SERP malgre les 301. Les Quality Signals sont encore associes a www/http.

**Cause** : Sitemap nouvelle URL pas soumis + aucune demande de changement d'adresse.

### 🚨 7. Brand search "allo maison" = 0 clicks sur 10 impressions (pos 2.4)

Pire ratio du dataset. Explication probable :
- Google affiche l'ancien `http://www.allo-maison.ma/` en SERP
- Meta description de l'ancien site poussiereuse
- Ou le concurrent `allomaison.fr` / `allopro.ma` intercepte le clic

**Action** : Reindexer `/` comme priorite absolue + forcer la transition domaine.

### ⚠️ 8. Chaine de redirect 2 hops

`http://www.allo-maison.ma/femme-de-menage/` →
  (1) 301 → `https://allo-maison.ma/femme-de-menage/` →
  (2) 308 → `https://allo-maison.ma/femme-de-menage-casablanca`

Chaque hop = dilution du PageRank + lenteur mobile. **Objectif : un seul 301**.

Fix : ajouter dans `next.config.ts` headers Netlify des redirects combinee http+www → canonical, OU gerer ca au niveau Cloudflare Page Rules.

### ⚠️ 9. Cloudflare ecrase robots.txt Next.js

Le robots.txt live contient 2 sections (Cloudflare Managed + Next.js), avec des contradictions :

| Bot | Cloudflare | Next.js |
|-----|------------|---------|
| ClaudeBot | Disallow | Allow |
| Applebot-Extended | Disallow | Allow |
| Bytespider | Disallow | Disallow |
| CCBot | Disallow | Allow |

**Cloudflare prime en SERP-robots interpretation**, donc ClaudeBot/Perplexity ne nous indexent pas. Or l'equipe veut clairement AI-friendly (cf. robots.ts Allow pour GPTBot, ClaudeBot, PerplexityBot).

**Action** : Desactiver "AI Audit / AI Bot blocking" dans Cloudflare (Security > Bots > AI Audit), OU supprimer la duplication.

### ⚠️ 10. `/logo.png` 404 mais reference partout en JSON-LD

Organization.logo pointe vers `/logo.png` qui retourne 404. Idem `/favicon.ico` (404).

**Impact** : Google Knowledge Graph ne peut pas afficher de logo brand + potentiels warnings "missing image" en Rich Results Test.

**Fix** : Soit convertir `/public/brand/logo-mark.svg` en `/public/logo.png` (PNG 512×512), soit changer la reference en `/brand/logo-lockup.svg`.

## 5 OPPORTUNITES MANQUEES (QUICK WINS)

### 💰 1. "Agence femme de ménage/nounou" = pos 1 avec 0% CTR

Top queries 28j avec position 1 ET 0 click :
- `agence femme de ménage casablanca` : 10 imp / pos 1.1 / **0 click**
- `agence nounou casablanca` : 11 imp / pos 1.0 / **0 click**
- `agence de ménage`, `agence ménage`, `agence nounou`, `agence menage casablanca` : 1-2 imp each / pos 1 / 0 click

**Diagnostic** : On rank #1 sur un mot qui n'est PAS dans notre title. Google pousse une autre page ou la meta est coupee.

Actuel : `Femme de Ménage Casablanca – Tarifs & Loi 19-12 | Allo Maison`

**Fix** : Changer en
```
Agence de Femme de Ménage Casablanca · Vérifiée & Loi 19-12 | Allo Maison
```
Et idem pour nounou :
```
Agence de Nounou Casablanca · Profils Vérifiés & Tarifs 2026 | Allo Maison
```

Gain estime : de 0% CTR a 5-8% = +1-2 clicks/jour x 6 villes x 2 services = ~15-25 leads/mois.

### 💰 2. "plombier/electricien/serrurier" pages = 0 impressions

Les pages `/urgence/*/*` (18 URLs) et `/plombier-*` / `/electricien-*` / `/serrurier-*` (~18 URLs) n'ont **AUCUNE impression** apres 4 jours. Pages pourtant pretes.

Causes probables :
1. Not indexed (cf. faille #2)
2. Grammar/quality signals (cf. faille #4)
3. Schema trop pauvre vs competition (cf. faille #5)

Apres fix 1/4/5, prevoir +500-1500 imp/mois sur ces 36 pages (d'apres keyword research : plombier casablanca = 1.3K/mois).

### 💰 3. Anciens blog posts perdus

GSC remonte encore des clicks sur des URLs que Google n'a pas reindexees :

Les 301 sont ok, mais :
1. Elles contribuent ~35 imp (11% du total) a un domaine qu'on abandonne
2. Google met ~3-6 mois a propager la canonical via 301

**Fix** : Soumettre "Change of Address" dans GSC (www → non-www) + relancer indexation API sur les nouvelles destinations.

### 💰 4. Meta titles pages urgence a refondre

Pattern actuel : `Plomberie Urgence Casablanca | Intervention 30 min | Allo-Maison`

Problemes :
- "Plomberie Urgence" = ordre mots non-naturel (cherche "plombier urgence")

Suggestion : `Plombier Urgence Casablanca 24/7 · Intervention 30min · Dès 250DH | Allo Maison`

### 💰 5. Schema Service + LocalBusiness sur toutes les pages city
